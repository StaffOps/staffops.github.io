What we build

agents

Aigent Squad

Multi-agent platform for AWS and Kubernetes operations. Supervisor routes queries to domain specialists. Read-only by default.

Python · FastAPI · Bedrock · K8s
orchestration

ChaiTops

Conversational orchestrator for observability stacks. LibreChat frontend with multi-CLI adapter support.

Python · LibreChat · Redis · OTel
detection

Anomaly Detection

Distributed anomaly detection for Kubernetes. Go workers query VictoriaMetrics and Loki, ML service correlates signals.

Go · Python · gRPC · VictoriaMetrics
core

Agent Definition

Source of truth for agent behavior. Steering rules, lazy-loaded skills, and subagent definitions. Works with Claude Code and Kiro.

Markdown · YAML · Bash
deploy

Helm Charts

Kubernetes-ready charts for the full suite. Auto-published via GitHub Actions on every release.

Helm · GitHub Actions · K8s

Philosophy

01 — open

Code, architecture decisions, and operational learnings are all public. No enterprise edition, no feature gating.

02 — collaborative

Contributions are welcome from anyone. We build on and give back to the open source ecosystem we depend on.

03 — free

No paywall, no SaaS lock-in. Every tool runs on your own infrastructure, under your control.

04 — pragmatic

Ship working solutions first, optimize second, document always. A working tool beats a perfect unshipped one.

Quick start

helm 
// add the chart repository
helm repo add staffops https://staffops.github.io/helm-charts/
helm repo update

// deploy aigent squad
helm install aigent-squad staffops/aigent-squad \
  --namespace staffops \
  --create-namespace
